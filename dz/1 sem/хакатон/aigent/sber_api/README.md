Сейчас файлы - работающие примеры взамодейстия с api gigachat

Для отправления запросов необходим файл access_token.txt в котором должен находиться токен доступа.
Для его формирования нужно запросить в ЛК developers.sber.ru ключ авторизации, записать данные в authorization_key.txt и запустить get_acces_token.py.
- В txt файл нужно записать client id, client secret и authorization key. Всё должно находиться на новой строчке соотвественно.
После записи скриптом get_acces_token.py файла authorization_key.txt можно (!но пока не нужно) отправить настоящий запрос к модели запустив ask.py.

Никакой из скриптов не принимает через командную строку данные, т.к. это пока что только примеры
